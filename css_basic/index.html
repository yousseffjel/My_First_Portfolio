Overview of HTML

HTML which stands for HyperText Markup Language, is the language used to describe the structure of documents which are meant to be displayed on the web. To this effect, any content that you see on a website has some HTML code behind it.

When you are asked to write a formal letter to your boss, which tool or software will you use to write the letter? You might have answered “Microsoft Word or Google Docs”. These tools are referred to as Word processors to text editors. The are meant to be used to write text like letters, etc.

In the same way, we need a tool that will allow us to write for the web. If I want to create a website and structure the content on it in my own way using HTML, where then will I write the code? How to use Elements and Attributes in your HTML code

Now that we know about tags, let’s explore more about elements and attributes and how they appear in our HTML code. Remember, an element in HTML is like a block or a piece of a puzzle that makes up part of our webpage. Each element has a specific job, whether it’s to display text, link to another page, show an image, and so on.

What is an Element?

An element is made up of several parts:

  • the opening tag,
  • the content, and
  • the closing tag.
For example, a simple text element looks like this:

The whole thing together, from the opening tag to the closing tag, is called an element.
